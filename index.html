📁 CSE Beginner: C Programming

১. C প্রোগ্রামের বেসিক স্ট্রাকচার +

একটি সাধারণ সি প্রোগ্রাম নিচের ফরম্যাটে শুরু হয়:

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

ব্যাখ্যা: #include লাইব্রেরি যুক্ত করে এবং main() থেকে প্রোগ্রাম চলা শুরু হয়।

২. ডাটা টাইপ এবং ভেরিয়েবল +

প্রোগ্রামে তথ্য রাখার জন্য ভেরিয়েবল ব্যবহার করা হয়:

int age = 22;      // পূর্ণসংখ্যা (Integer)
float result = 3.5; // দশমিক (Float)
char sign = 'A';   // অক্ষর (Character)
৩. ইনপুট এবং আউটপুট +

ইউজার থেকে তথ্য নিতে এবং দেখাতে ব্যবহৃত হয়:

printf("Enter Number: ");
scanf("%d", &num); // ইনপুট নেওয়া
printf("Result is: %d", num); // আউটপুট দেখানো
